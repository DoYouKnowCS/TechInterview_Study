# 📚 기술면접 대비 공부 진행 방식

## 💡 기획의도
- 기존에 진행한 CS스터디에서 배운 지식들을 몸에 익히기 위함.
- 면접 진행전 정리노트처럼 활용할 예정.

## 📂 폴더구조
- [CS분야폴더] - [단원이름.md파일]
- 예시) [Network] - [CDN.md]

## 😀 Commit Message Template
- [본인이름이니셜]: 단원명 | 세부단원 내용
- 예시) git commit -m "[LSD/LYJ]: Network | CDN의 동작원리 "

## ➕ 추가사항
- 꼭 기술관련부분이 아니여도 됨.
- 예시) [AWS] - [Route53.md] 처럼 자신이 개발하면서 배운 것도 올려도 됨!!!

## ⚠️ 주의사항
- 확실하게 찾은/공부한 정보만 올릴것!!!!
- 개발지식과 CS를 확실하게 구분지을것!!!
